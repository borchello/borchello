## Runner Terminal

A real-time prediction market terminal built on [Polymarket](https://polymarket.com).

**What's inside:**

- Live CLOB order execution via Polymarket API
- Copy trading engine with Privy Server Signers (non-custodial)
- AA wallets via ZeroDev (ERC-4337)
- Real-time WebSocket + SSE data feeds
- AI-powered Insights with SSR meta & JSON-LD schema
- Interactive 3D globe (globe.gl + Three.js) with signal visualization
- Portfolio analytics, leaderboard, audit ledger
